# Mohamed Saif - Portfolio Website

A modern, interactive portfolio website showcasing frontend development skills, projects, and experience.

## Features

- **Responsive Design** - Mobile-first approach with smooth responsiveness
- **Interactive Hero Section** - Horizontal scrolling hero with animations
- **Parallax Effects** - Engaging visual effects with CSS parallax
- **Project Showcase** - Display of personal projects with links
- **Tech Stack Display** - Visual representation of technical skills
- **Smooth Navigation** - Anchor-based smooth scrolling
- **Mobile Menu** - Hamburger navigation for mobile devices
- **Performance Optimized** - Optimized for fast loading and great performance

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mohamedsaif21/saif_portfolio_.git

# Navigate to project directory
cd saif_portfolio_

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for easy deployment on Vercel:

1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect the Vite configuration
3. Click "Deploy"

### Manual Deployment

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel
```
